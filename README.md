# Dijkstra Algorithm
## Input type
Dictionary

    "A": {"B": 5, "D": 3},
    "B": {"C": 1},
    "C": {"D": 1},
    "D": {"B": 2}
    
## How to use
Type file name, for example:

    graph2.json
    
Then input node's, for example: A and E
AOutput will be optimal path length.
